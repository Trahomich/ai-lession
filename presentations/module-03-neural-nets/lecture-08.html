Свёрточные нейросети (CNN)

Как ИИ «видит» картинки

Лекция 8 · Модуль 3. Нейросети — от нейрона до глубокого обучения

Почему обычная сеть не подходит?

Размер данных

Фото 1000×1000 RGB = 3 млн чисел

Первый скрытый слой (1000 нейронов) = 3 млрд весов

Только на одном слое!

Потеря структуры

Полносвязная сеть не знает, что пиксели — соседи

Для неё все пиксели — одинаковые числа

Как читать книгу с перемешанными буквами

Изображение = массив чисел

Каждый пиксель — три числа (RGB) от 0 до 255

Пиксель: (R=255, G=128, B=0) → оранжевый цвет

Фотография с телефона: 12 млн пикселей = 36 млн чисел

аналогия

Человек видит не пиксели — он видит линии, формы, объекты. Зрительная кора работает от простого к сложному.

Свёртка — поиск узоров

Маленькая матрица (фильтр) скользит по изображению и находит паттерны

Как работает

  • Фильтр 3×3 накладывается на фрагмент
  • Каждый пиксель × вес фильтра
  • Всё складывается → одно число
  • Сдвиг на 1 пиксель → повтор

Фильтр «вертикальный край»

-1 0 1
-1 0 1
-1 0 1

Даёт большое значение там, где вертикальная граница

Иерархия признаков

1
Первые слои — края, линии, градиенты
2
Средние слои — текстуры, формы, части объектов
3
Глубокие слои — объекты: глаза, колёса, лица, цифры
ключевой принцип

CNN строит представление от простого к сложному — как зрительная кора человека

Карты признаков

Один фильтр → одна карта признаков. 64 фильтра → 64 карты.

Вход: 32×32×3 (RGB) → После Conv(64): 32×32×64

Каждая карта показывает где на изображении найден конкретный узор

аналогия

64 «фонарика», каждый подсвечивает свой тип узора на картинке

Параметры свёртки

Stride (шаг)

На сколько пикселей сдвигается фильтр

Stride 1 → тот же размер

Stride 2 → размер ×0.5

Padding (дополнение)

Нули по краям изображения

«Same» → выход = вход по размеру

«Valid» → без дополнения, меньше

Dilation (расширение)

«Раздвигание» фильтра с дырками

Охватывает большую область

Без увеличения параметров

Количество фильтров

Определяет глубину выхода

32, 64, 128, 256 — типичные значения

Больше фильтров = больше признаков

Пулинг — сжатие с сохранением сути

Max Pooling: берём максимум из окна 2×2

6 2 → 6
3 8 → max(6,2,3,8) = 8

Зачем

  • Меньше данных → быстрее
  • Меньше параметров → меньше overfitting
  • Инвариантность к сдвигам

Аналогия

Смотреть на пейзаж через трубочку и запоминать только самую яркую точку в каждом участке

Архитектура CNN

Входное изображение
→ Conv → ReLU → Pool
→ Conv → ReLU → Pool
→ Conv → ReLU → Pool
→ Flatten (разворачиваем в вектор)
→ Fully Connected → ReLU
→ Fully Connected → Softmax
→ Ответ: «кот — 94%, собака — 5%, птица — 1%»

Каждый блок: меньше размер, больше признаков

От признаков к решению

Flatten

«Стопка» карт признаков → один вектор

Например: 7×7×512 = 25 088 чисел

Softmax

Превращает числа в вероятности

Сумма всех выходов = 1 (100%)

«Я на 94% уверен, что это кот»

Знаменитые архитектуры

98
LeNet-5 — Ян ЛеКун, распознавание цифр на чеках. 5 слоёв, 60K параметров
12
AlexNet — прорыв! 8 слоёв, 60M параметров, GPU. ImageNet с огромным отрывом
14
VGG — простота: все фильтры 3×3, просто много слоёв (16–19)
15
ResNet — 152 слоя! Skip connections → информация «перепрыгивает» через слои

Применение: распознавание лиц

CNN находит лицо → извлекает ключевые точки → сравнивает с эталоном

реальность

Разблокировка телефона, безопасность, массовое наблюдение

Применение: медицина

  • Анализ рентгеновских снимков
  • Обнаружение рака кожи по фото
  • Чтение МРТ и гистологических препаратов
  • В ряде задач — превосходит врачей
важно

ИИ — не замена врачу, а «вторая пара глаз». Подсвечивает подозрительные участки, ускоряет диагностику.

Применение: автопилот и поиск

Автопилот

Камеры анализируют дорогу десятки раз в секунду

Пешеходы, машины, знаки, разметка

Google Photos

Поиск «закат на пляже» — без единого тега

CNN проанализировала каждое фото и создала визуальный индекс

Transfer Learning

«Стоя на плечах гигантов»

идея

Кто-то потратил миллионы долларов и месяцы, обучив ResNet на миллионах изображений. Ранние слои уже умеют находить края и формы.

Вы берёте предобученную модель, «замораживаете» ранние слои и дообучаете только последние под свою задачу.

Вместо месяцев → часы · Вместо млн изображений → тысячи · Вместо кластера GPU → одна карта

Демонстрация

вживую

Что «видит» CNN

  • Фильтры 1 слоя — края, линии, цвета
  • Глубокие слои — лица, текстуры, объекты
  • Grad-CAM — тепловая карта: на что сеть смотрела при решении

Ключевые идеи

  • Изображение = массив чисел (RGB, пиксели)
  • Свёртка — фильтр скользит по картинке, ищет узоры
  • Иерархия: края → формы → текстуры → объекты
  • Пулинг — сжатие данных с сохранением сути
  • Transfer learning — не обучаем с нуля
  • CNN уже повсюду: лица, медицина, автопилот, поиск фото
