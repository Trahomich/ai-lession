Лекция 5
Линейная регрессия: y = w₁x₁ + w₂x₂ + b
Градиентный спуск
Модель учится, корректируя веса
Классификация
Сигмоида: число → вероятность

Проблема: линейная модель проводит только одну прямую. А мир — нелинеен.

Секция 1

Биологический нейрон — упрощённо

🟢 Дендриты

Входы нейрона. Получают сигналы от других нейронов. Тысячи связей.

🔵 Сома (тело)

Суммирует все сигналы. Принимает решение — активироваться или нет.

🟣 Аксон

Выход нейрона. Передаёт импульс дальше к другим нейронам.

Синапсы — места стыковки. Их сила меняется при обучении. Это и есть «память» мозга.

Ключевое наблюдение

Обучение — это изменение силы связей между нейронами

Человеческий мозг: ~86 миллиардов нейронов. Каждый нейрон — простая единица. Интеллект возникает из взаимодействия миллиардов.

⚠️ Мы НЕ моделируем мозг точно. Искусственный нейрон берёт только самую суть: входы → веса → сумма → порог.

Секция 2

Искусственный нейрон — перцептрон

1943: Маккалл и Питтс — математическая модель нейрона

1957: Фрэнк Розенблатт — перцептрон, первая обучаемая нейросеть

z = w₁·x₁ + w₂·x₂ + w₃·x₃ + b

y = активация(z)

Три шага нейрона

Шаг 1
Входы × веса
Каждый вход x умножается на свой вес w
Шаг 2
Суммирование
z = w₁x₁ + w₂x₂ + ... + b
b — bias (порог)
Шаг 3
Активация
y = f(z)
Нелинейная функция

Bias — насколько легко нейрону «активироваться». Аналогия: начальный уровень воды в чаше.

Пример: «Ити на прогулку?»

Входы:

  • x₁ = температура (20°C)
  • x₂ = дождь? (0/1)
  • x₃ = свободное время? (0/1)

Веса:

  • w₁ = 0.5 (тепло — хорошо)
  • w₂ = -0.8 (дождь — плохо)
  • w₃ = 0.3 (время — полезно)
z = 0.5·20 + (-0.8)·0 + 0.3·1 + (-5) = 10 + 0 + 0.3 − 5 = 5.3
y = активация(5.3) → Идём гулять!

Функции активации

Ступенчатая (step)

z > 0 → 1, иначе → 0

Простая, но «рваная» — нельзя применить градиентный спуск.

Сигмоида (sigmoid)

σ(z) = 1/(1 + e⁻ᶻ)
Плавная, 0→1. Но на краях градиент ≈ 0 — обучение «застревает».

ReLU ⭐ стандарт

z > 0 → z, иначе → 0

Простая, быстрая, не «застревает». Используется в 95% нейросетей.

Зачем вообще?

Без активации нейрон = линейная регрессия!

Активация добавляет нелинейность — способность моделировать сложные зависимости.

Секция 3

Задача XOR — почему одного нейрона мало

XOR (исключающее ИЛИ):

  • 0 XOR 0 = 0
  • 0 XOR 1 = 1
  • 1 XOR 0 = 1
  • 1 XOR 1 = 0

Попробуйте провести одну прямую, которая отделит нули от единиц.

○(0,0)    ●(0,1)

●(1,0)    ○(1,1)

Невозможно провести прямую!

Последствия

1969
Минский и Паперт: «Перцептроны» — один нейрон не решает XOR
1970-е
«Зима ИИ» — финансирование сокращено, разочарование
Но!
Многослойные сети МОГУТ решить XOR. Просто не было алгоритма обучения.
1986
Backpropagation — алгоритм для обучения многослойных сетей

Теорема универсальной аппроксимации (1989)

Нейросеть с одним скрытым слоем (но достаточным количеством нейронов) может аппроксимировать любую непрерывную функцию с любой заданной точностью.

Грубо говоря: достаточно нейронов → можно нарисовать ЛЮБУЮ границу.

⚠️ Теорема говорит что это ВОЗМОЖНО, но не КАК найти правильные веса. На практике нужны несколько слоёв.

Итоги

Что мы узнали

  • Биологический нейрон: дендриты → сома → аксон. Обучение = изменение синапсов.
  • Искусственный нейрон: входы × веса + bias → активация → выход
  • Функция активации — ключ к нелинейности. Без неё = линейная регрессия.
  • ReLU — стандарт в современных нейросетях
  • Один нейрон = одна прямая. XOR — невозможно.
  • Решение: соединить нейроны в сеть — следующая лекция!
