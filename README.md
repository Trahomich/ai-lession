# Искусственный интеллект: от истоков до наших дней

Курс лекций по истории и основам искусственного интеллекта.

## Цель

Устранение магизма — понимание того, как реально работают нейросети и ИИ, через призму истории их развития. Без мистики, без хайпа, только факты и логика.

## Для кого

Для широкой аудитории — от любопытных гуманитариев до специалистов, которые хотят увидеть картину целиком.
